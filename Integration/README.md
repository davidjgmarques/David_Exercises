## Language

Python3

## Libraries necessary

numpy 
matplotlib
scipy

## Usage

Windows 10

```bash
py ./Dav_integrators.py
```

Unix/MacOS

```bash
python ./Dav_integrators.py
```

Note:
If several versions of python are installed, the initialization is made with 'python3' instead of 'python'
